Estructura de CSS


En CSS, las reglas están compuestas por selectores, propiedades y valores.


  1. Selector:

    2. El selector apunta al elemento HTML al que se aplicarán los estilos. Puede ser un tipo de elemento, una clase, un identificador, etc.



  2. Propiedad:

    3. La propiedad es el aspecto específico del estilo que se va a aplicar al elemento seleccionado.



  3. Valor:

    4. El valor es la configuración específica que se asigna a la propiedad.

¿Qué es el modelo de cajas en CSS?

El modelo de cajas en CSS describe cómo se representan y se manejan los elementos en una página web. Cada elemento HTML es considerado como una caja rectangular con propiedades como margen, borde, relleno y contenido.


Elementos de línea y elementos de bloque

  1. Elementos de Bloque:

    2. Son elementos que ocupan toda la anchura disponible y comienzan en una nueva línea.



  2. Elementos de Línea:

    3. Son elementos que ocupan solo el ancho necesario y no inician una nueva línea.
